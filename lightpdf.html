1. Lightpdf


      2. LightPDF is an online PDF tool that provides various PDF-related features, such as converting PDF files to other formats like Word, Excel, PowerPoint, JPG, PNG, and vice versa.
      It also allows users to merge, split, compress, and unlock PDF files, as well as add watermarks, page numbers, and annotations. Users can access LightPDF's services through its
      website or download its desktop version for offline use. LightPDF offers a free version with basic features, as well as a premium version with additional features and no ads. Overall, LightPDF is a useful tool for anyone who frequently works with PDF files and needs a quick and easy way to edit or convert them.

  2. Printfriendly

    3. "PrintFriendly" is a tool that allows users to make a web page more printer-friendly or PDF-friendly by removing unnecessary elements such as ads, navigation menus, and other distracting elements. It can be used by clicking on the "Print Friendly" button or by installing a browser extension that allows you to use the tool directly in your browser. Once the tool is activated, you can modify the content of the page by deleting sections, resizing images, and changing the font size, among other things, to create a cleaner, more printable version of the web page.
